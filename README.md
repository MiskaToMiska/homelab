# Homelab Infrastructure

Self-hosted homelab for fun, learning, and experience. Fully automated using Ansible and Docker Compose.

## 💻 Hardware

- **Base:** HP EliteDesk 800 G3 Mini
- **CPU:** Intel Core i5-7500 (4 Cores)
- **RAM:** 1x16GB DDR4
- **Storage:** 2x 256GB NVMe M.2 SSD
- **OS:** Debian 13 Trixie

## 📦 Current Stack
- Core: Pi-hole (DNS/AdBlock) & Caddy (Reverse Proxy + automated DuckDNS TLS)
- Upcoming apps: Jellyfin, Kavita, Swing-music, Paperless-ngx, Stirling-PDF, Grafana, Prometheus, Home Assistant.
