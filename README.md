# Homelab infrastructure

Self hobby hosted homelab for fun and experience

## Hardware

- BASE: HP Elitedesk 800 g3
- CPU: Intel Core i5-7500
- RAM: 1x16GB DDR4
- DISKS: 2x256GB NVMe M.2 SSD

## Stack

- Debian 6.12
- Podman
- Tailscale VPN
- Jellyfin + Arr-stack
- Kavita
- Swing-music
- Paperless
- Grafana + Prometheus + Dozzle
- Home assistant
- Pihole + nginx-proxy-manager
- Vaultwarden
- and many more...
